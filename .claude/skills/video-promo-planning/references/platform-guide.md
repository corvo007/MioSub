### Algorithm Factors (Priority Order)

1. **完播率** (Completion rate) - Most important metric
   - Early drop-offs (first 1-2 seconds) signal hook failure
   - Replays are strongest positive signal
2. **互动率** (Engagement) - Likes, coins, favorites, shares, comments
3. **弹幕密度** (Danmaku density) - Unique to B站, signals real-time engagement
4. **投币** (Coins) - Strong quality signal, users have limited daily coins
5. **收藏** (Favorites) - Indicates long-term value
6. **转发** (Shares) - Viral potential indicator

### 弹幕 (Danmu) Strategy

**What makes B站 unique:**

- Time-synced comments overlay directly on video as it plays
- Creates "watching with friends" social experience
- Comments are cumulative - builds community over time
- Can be toggled on/off by viewers

**How to leverage:**

- Design "弹幕互动点" - moments that invite comments
- Ask questions: "你们觉得这个功能怎么样？"
- Create shareable moments: "弹幕告诉我你用的是什么工具"
- Leave space for danmu (don't fill entire screen with text)

### Content Style

- **Personality matters** - B站 users follow creators, not just content
- **Longer acceptable** - 10-15 min videos perform well if engaging
- **Community references** - Appropriate memes and 网络用语
- **Educational depth** - Users appreciate thorough explanations
- **Authenticity** - "像建设者一样说话，不是营销人员"
- **Niche focus** - Platform surfaces specific content, not just viral

### 三连文化 (Sanlian Culture)

**Like + Coin + Favorite** is a ritual on B站:

- Coins are free daily currency users give to appreciate content
- Best timing: After delivering significant value, not at the start
- Don't be pushy - earned through quality, not begging
- Example: "如果这个视频对你有帮助，三连支持一下"

### Best Practices

1. **开头不要废话** - Skip generic intros, get to value in first 10 seconds
2. **设置弹幕互动点** - Design moments for danmu engagement
3. **三连提醒** - Remind after value delivery, not before
4. **分P考虑** - Consider multi-part for very long content (>15 min)
5. **封面文字** - Chinese text, high contrast, emotional expression
6. **合集功能** - Group related videos for series discovery
7. **专栏配合** - Write companion articles for SEO and depth

### B站 Thumbnail (封面) Design

**Key principles:**

- **Chinese text** - Use Chinese, not English
- **High contrast** - Readable at small size in feed
- **Emotional expression** - Face with expression if possible
- **3 elements max** - Main subject + object + minimal text
- **Curiosity trigger** - Show enough to spark interest, not everything

**What works on B站:**

- Before/after comparisons
- Surprised or curious expressions
- Clear product/result preview
- Bold, contrasting colors
- Text that completes the hook

- Emotional words work: 太强了、绝了、神器
- Avoid pure clickbait - B站 users value authenticity

## YouTube Optimization

### Key Statistics

| Metric             | Data                                    |
| ------------------ | --------------------------------------- |
| Daily Shorts views | 70 billion                              |
| Silent viewing     | 83% watch with sound off                |
| Caption completion | 80% more likely to finish with captions |
| Thumbnail decision | 10-20 milliseconds                      |
| Face visibility    | 11x more noticeable                     |
| Long-tail views    | 40% happen after 30 days                |

### Algorithm Factors (Priority Order)

1. **Click-through rate (CTR)** - Thumbnail + title effectiveness
2. **Watch time** - Total minutes watched (not just views)
3. **Audience retention** - Percentage watched, especially first 30 seconds
4. **Session time** - Do viewers stay on YouTube after your video?
5. **Engagement** - Likes, comments, shares, subscribes

### Content Style

- **Hook immediately** - First 30 seconds critical for retention
- **Faster pacing** - Tighter editing expected vs B站
- **SEO-conscious** - Keywords in title, description, tags
- **Chapters** - Use timestamps for navigation and SEO
- **Pattern breaks** - Change visuals every 5-10 seconds

### Best Practices

1. **Pattern interrupt** - Unexpected visual/sound in first 3 seconds
2. **Open loops** - Tease upcoming content to retain viewers (+32% watch time)
3. **End screens** - Use last 20 seconds for cards/end screens
4. **Playlists** - Group related videos for session time
5. **Thumbnail A/B** - Test different thumbnails (YouTube Studio feature)

### Optimal Posting Times (UTC)

| Day      | Best Times  | Notes                           |
| -------- | ----------- | ------------------------------- |
| Weekdays | 14:00-16:00 | Catches US morning + EU evening |
| Weekends | 09:00-11:00 | Leisure browsing time           |

**Important**: Consider your target audience timezone.

### SEO Checklist

- [ ] Primary keyword in title (front-loaded, first 60 characters)
- [ ] Keyword in first 2 sentences of description
- [ ] 200+ word description with natural keyword usage
- [ ] 5-8 relevant tags (most important first)
- [ ] Custom thumbnail (1280x720, <2MB, high contrast)
- [ ] Chapters with timestamps (first must be 00:00)
- [ ] Cards linking to related content
- [ ] End screen with subscribe + video suggestions
- [ ] Captions/subtitles (indexed by algorithm)

## Cross-Platform Strategy

### Content Adaptation Matrix

| Element           | B站                       | YouTube                  |
| ----------------- | ------------------------- | ------------------------ |
| Intro length      | 15-30s acceptable         | 5-15s max                |
| Pacing            | Moderate, can breathe     | Fast, tight edits        |
| Personality       | More casual, 网络用语 OK  | More polished            |
| CTA style         | 三连 (like/coin/favorite) | Subscribe + bell         |
| Language          | Chinese                   | English (or localized)   |
| Captions          | 推荐                      | 必须 (83% muted)         |
| Video length      | 8-15 min sweet spot       | 5-10 min sweet spot      |
| Shorts equivalent | 竖屏视频 (<3 min)         | Shorts (<60s)            |
| Thumbnail text    | Chinese, emotional        | English, benefit-focused |
| Community         | 弹幕 interaction          | Comments section         |

### Repurposing Workflow

1. **Create master version** - Full content, neutral style, all features covered
2. **B站 version** - Add personality, Chinese captions, 弹幕 interaction points, longer intro OK
3. **YouTube version** - Tighter edit, English captions, SEO optimization, chapters
4. **Shorts/竖屏** - Extract 60s highlights, vertical format, loop-friendly

### Content Compounding Strategy

**Long-term thinking:**

- 40% of views happen after 30 days
- Each repeat video on same topic improves CTR by ~10%
- Build a library, not just campaigns
- Series > one-off videos

**Series Design:**

- "Viewers subscribe for consistency, not variety"
- Choose 1-2 repeatable formats
- Create topic clusters, not isolated videos
- Link videos together with cards and end screens

### Common Cross-Platform Mistakes

| Mistake                       | Impact                      | Fix                                  |
| ----------------------------- | --------------------------- | ------------------------------------ |
| Same thumbnail for both       | Lower CTR                   | Adapt text language, style           |
| Same pacing                   | Poor retention              | Tighten for YouTube, breathe for B站 |
| Ignoring captions             | Lose 83% of YouTube viewers | Always add captions                  |
| Generic titles                | Poor SEO                    | Localize keywords per platform       |
| One-and-done posting          | Miss long-tail              | Promote over weeks, not just day 1   |
| No 弹幕 design                | Miss B站 engagement         | Plan interaction moments             |
| English text on B站 thumbnail | Lower CTR                   | Use Chinese text                     |
