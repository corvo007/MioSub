---
title: 'Video Download'
---

Download videos from YouTube and Bilibili with built-in yt-dlp engine.

---

## ✅ Supported Links

| Platform     | Type       | Example                        |
| :----------- | :--------- | :----------------------------- |
| **YouTube**  | Standard   | `youtube.com/watch?v=xxx`      |
|              | Short URL  | `youtu.be/xxx`                 |
|              | Shorts     | `youtube.com/shorts/xxx`       |
|              | Embed      | `youtube.com/embed/xxx`        |
| **Bilibili** | BV/av ID   | `bilibili.com/video/BVxxx`     |
|              | Multi-part | `bilibili.com/video/BVxxx?p=2` |
|              | B23 Short  | `b23.tv/xxx`                   |

---

## ❌ Not Supported

| Platform | Type               | Reason                     |
| :------- | :----------------- | :------------------------- |
| YouTube  | Playlists/Channels | Use individual video links |
| Bilibili | Anime/Movies       | Copyright restrictions     |
|          | Paid Courses       | Requires purchase          |
|          | Livestreams        | Real-time streams          |
|          | Premium/Charged    | Requires login cookies     |
|          | Favorites/Profile  | Use individual video links |
