Welcome to MioSub! This guide will take you from installation to generating your first professional subtitle in **under 5 minutes**.

<Callout type="info" title="System Requirements">
  - Windows 10/11 (64-bit)
  - 4GB+ available RAM
  - Internet connection (for API calls)
</Callout>

---

## Step 1: Download & Install

<Steps>
  <Step title="Get the Installer">
    Visit [GitHub Releases](https://github.com/corvo007/Gemini-Subtitle-Pro/releases) and download the latest `MioSub-x.x.x-win-x64.zip`
  </Step>
  <Step title="Extract & Run">
    Extract to any location (recommended: `D:\Apps\MioSub`) and double-click `MioSub.exe`
  </Step>
  <Step title="First Launch">
    On first launch, MioSub will create configuration files automatically. Wait for the UI to load.
  </Step>
</Steps>

---

## Step 3: Generate Your First Subtitle

Now let's create your first professional subtitle!

<Steps>
  <Step title="Select Video">
    **Option A**: Click "Open File" to select a local video
    
    **Option B**: Paste a YouTube/Bilibili URL for auto-download
  </Step>
  <Step title="Choose Languages">
    - **Source Language**: Original audio language (auto-detect supported)
    - **Target Language**: Translation target (e.g., English)
  </Step>
