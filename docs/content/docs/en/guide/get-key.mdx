## â“ FAQ

### "Region not supported" error?

Gemini API has regional restrictions. Ensure your network can access Google services.

### Can I use multiple Google accounts to increase quota?

Yes! MioSub supports multiple API Keys (comma-separated). The system will automatically rotate between them.

### Does the API Key expire?

No! Free tier keys are permanent unless you manually delete them.

### Is the free quota enough?

Absolutely for personal use:
- **Free quota**: 15 requests per minute
- **Typical usage**: ~50-100 requests for a 30-minute video
- **In practice**: 10+ videos per day, more than enough
