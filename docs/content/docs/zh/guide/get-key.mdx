## 🚀 获取步骤

<Steps>

<Step>
### 访问 Google AI Studio

打开浏览器，访问 [Google AI Studio](https://aistudio.google.com/app/apikey)

> 如果提示登录，使用你的 Google 账号登录即可。
</Step>

<Step>
### 创建 API Key

1. 点击左上角的 **「Get API Key」** 按钮
2. 在弹出的窗口中点击 **「Create API key in new project」**
3. 等待几秒钟，系统会自动生成一个 API Key
</Step>

<Step>
### 复制你的 Key

生成成功后，你会看到一个以 `AIza` 开头的长字符串，这就是你的 API Key。

点击复制按钮，保存好这个 Key。

<Callout type="warning">
**注意**：这个 Key 只会显示一次！请立即复制保存。如果忘记了，可以删除重新创建。
</Callout>
</Step>

<Step>
### 填入 MioSub

1. 打开 MioSub 应用
2. 点击右上角 **「设置」** ⚙️
3. 找到 **「服务」→「Gemini API Key」**
4. 粘贴刚才复制的 Key
5. 点击保存

🎉 **大功告成！** 现在你可以开始使用 MioSub 了！
</Step>

</Steps>

## ❓ 常见问题

### 提示 "地区不支持" 怎么办？

Gemini API 在部分地区有限制。请确保你的网络环境可以正常访问 Google 服务。

### 我有多个 Google 账号，可以叠加额度吗？

可以！MioSub 支持填入多个 API Key（用逗号分隔），系统会自动轮询使用，有效提高并发上限。

### API Key 会过期吗？

不会！免费额度的 Key 是永久有效的，除非你手动删除。

### 免费额度够用吗？

对于个人用户来说，完全够用：
- **免费额度**：每分钟 15 次请求
- **实际使用**：处理一个 30 分钟视频大约需要 50-100 次请求
- **等效**：每天可以处理 10+ 个视频，完全够用
