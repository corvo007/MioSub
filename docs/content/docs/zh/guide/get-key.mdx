## 🚀 获取步骤

<Callout type="info">
考虑到国内的网络环境，以及官方API需要绑定信用卡，因此本教程使用 [云雾 API](https://yunwu.ai/register?aff=wmHr) 进行演示。
</Callout>

<Steps>

<Step>
### 注册

    打开浏览器，访问 [云雾 API](https://yunwu.ai/register?aff=wmHr) 进行注册。

    <Callout type="info">
此中转站无需信用卡、无需翻墙、无最低充值限制。是我个人很多家中转体验下来体验比较好的一个。当然，你也可以使用其他的中转站，或者是官方API（如果你Google Cloud 300美元额度还没用完的话）。
</Callout>

</Step>

<Step>
### 创建 API Key

1. 点击左边的 **「API 令牌」** 菜单
2. 点击 **「添加令牌」**
3. 按下方图片配置，选择一个最便宜的分组和一个比较稳定的分组，保证又便宜又稳。  

<img
    src="/apikey.png"
    alt="配置API Key 图片"
    style={{ maxWidth: '400px', borderRadius: '12px' }}
  />

</Step>

<Step>
### 复制你的 Key

生成成功后，把界面向右滚动，能看到密钥一栏，点击复制按钮，就得到了 API Key，保存好这个 Key。

</Step>

<Step>
### 填入 MioSub

1. 打开 MioSub 应用
2. 点击右上角 **「设置」** ⚙️
3. 找到 **「服务」→「Gemini API 密钥」**
4. 粘贴刚才复制的 Key
5. 填写 Gemini 端点，注意只需要填写域名（如：https://yunwu.zeabur.app 或 https://api3.wlai.vip）
5. 点击保存

🎉 **大功告成！** 现在你可以开始使用 MioSub 了！
</Step>

</Steps>
